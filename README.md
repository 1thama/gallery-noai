# Gallery No-AI 🖼️

Aplikasi galeri Android yang cepat, ringan, dan berfokus pada privasi untuk mengelola foto dan video secara lokal, murni tanpa integrasi kecerdasan buatan (AI).

## ✨ Fitur Utama

* **Tampilan Galeri & Album:** Jelajahi media dalam bentuk *grid* yang responsif atau kelompokkan berdasarkan album/folder.
* **Pencarian Cepat:** Temukan media dengan mudah, dilengkapi dengan riwayat pencarian lokal.
* **Manajemen Folder:** Pindahkan media atau buat folder baru langsung dari aplikasi.
* **Media Viewer Interaktif:** Tampilan detail media yang mendukung *zoom* (cubit untuk memperbesar).
* **Tempat Sampah (Trash):** Media yang dihapus akan masuk ke tempat sampah sebelum dihapus permanen.
* **Fast Date Scroller:** Navigasi super cepat menyusuri foto berdasarkan tanggal.

## 🛠️ Teknologi yang Digunakan

* **Bahasa:** Kotlin
* **UI Framework:** Jetpack Compose (Material Design)
* **Database:** Room (SQLite) untuk *caching* media
* **Arsitektur:** MVVM dengan Coroutines & Flow

## 🚀 Cara Menjalankan Proyek

1. *Clone* repositori ini ke komputer Anda:
```bash
git clone https://github.com/1thama/gallery-noai

```

2. Buka proyek tersebut menggunakan **Android Studio** versi terbaru.
3. Tunggu hingga proses *sync* Gradle selesai.
4. Klik tombol **Run** (`Shift + F10`) untuk membangun dan menjalankan aplikasi di perangkat atau Emulator Android.
